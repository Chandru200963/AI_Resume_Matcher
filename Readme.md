# AI-Based Resume Matcher

A Python-based NLP project that compares resumes with job descriptions
using TF-IDF vectorization and cosine similarity.

## Features
- Text preprocessing
- Stopword removal
- Vector-based similarity calculation
- Match percentage generation

## Tech Stack
- Python
- Scikit-learn
- TF-IDF
- Cosine Similarity

## How to Run
pip install -r requirements.txt
python resume_matcher.py
